# outcome/

- `语言文字规范_附录A_语法等级大纲.md`: raw OCR output (RapidOCR, no post-processing), see `docs/benchmark_result.md`.
- `processed.md`: the same content after heading/boilerplate/formatting cleanup, see `docs/process.md` for the exact rules applied.

Both files are a digitized transcription of Appendix A (语法等级大纲, pages
176-260) of **GF 0025-2021**, the *Chinese Proficiency Grading Standards for
International Chinese Language Education* (国际中文教育中文水平等级标准),
issued by China's Ministry of Education and National Language Commission.
